# Agent Skill Genome Dossier

Date: 2026-06-05

## Goal

Make SkillProposal feel like a replayable Worldline product asset rather than a static candidate card.

## In Scope

- Enrich local SkillProposal preview data with branch, gate, artifact, promotion, and acceptance metadata.
- Make skill cards focusable from the Skill Genome panel.
- Expand Skill Dossier with score, version, promotion state, acceptance criteria, permissions, gates, artifacts, and source episodes.
- Validate the `/worldline/agent` browser path with screenshot evidence.

## Out of Scope

- No backend schema change.
- No automatic skill creation without admin login.
- No new frontend dependencies.
