# 主题模块能力控制台对齐

## 目标

- 继续完善主题分区自定义模块，让每个模块能清楚展示已绑定知识库和可调用的 Worldline 后端能力。
- 页面保持简约，主要信息在一屏内可读，细节通过按钮、折叠区或抽屉查看。
- 全部面向用户的新增文案使用中文。

## 边界

- 本阶段只改前端主题模块体验和任务记录，不改后端 API、数据库 schema 或 MCP contract。
- 后端暂不可用时，页面仍应能用本地配置渲染，并清楚标注“待绑定/待接入”状态。
- 不做大范围视觉重写，不回退当前工作树中已有的其他改动。

## 验收

- 主题列表卡片能简洁展示模块绑定知识库、启用面和后端能力数量。
- 创建/编辑模块弹窗能展示后端能力映射和保存 payload 预览。
- 主题详情页能集中展示 Wiki、图谱、时间线、证据、MCP、工作流、质量门禁等后端入口。
- Vite 构建通过，并完成主题页浏览器截图/DOM 验证。
