# 设计

## 知识库详情图谱模块

- 顶部覆盖层保留搜索、刷新、设置按钮。
- 新增状态摘要：节点、边、来源接口、最后加载时间。
- 画布中间使用轻量状态卡：
  - 加载中：说明正在读取图谱。
  - 错误/降级：展示后端原因，提供重试。
  - 空图谱：说明需要先完成 LightRAG 文档入库或图谱生成。
  - 非 LightRAG：展示不可用原因。

## 数据流

- `GET /api/graph/subgraph` 仍是主图谱数据来源。
- `GET /api/graph/stats` 作为统计补充，失败不阻塞主图谱。
- 后端 `degraded_reason`、`message`、`available`、`degraded` 都进入统一错误/降级展示。

## 风险

- 当前后端不可用，QA 主要通过 mock API 验证状态机和 payload。
- G6 画布渲染本身不是本阶段改动重点，只验证状态层不遮挡主要操作。
