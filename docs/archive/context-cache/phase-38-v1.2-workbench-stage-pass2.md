# phase-38 v1.2 世界线工作台舞台增强第二波

## 本轮结论

- `v1.2` 第二波前端实现已完成，世界线工作台从“可用骨架”提升为“具有舞台感的工作台”。
- 本轮没有重做首页，没有替换基础聊天页，也没有把工作台塞进聊天页附属边栏。
- 当前工作台仍然是独立页面，但已经具备更明确的头部表达、分支树层次、当前分支焦点和右侧信息面板结构。

## 本轮实际增强

### 1. 工作台整体舞台感

- 强化了工作台头部与引导区，使页面首屏更像“展开未来路径”的舞台，而不是普通表单页。
- 强化了工作台中段的概览区、主线聚焦区和分支舞台区层次。
- 保留基础聊天界面不变，工作台继续作为独立主舞台存在。

### 2. 分支树与节点焦点

- 强化了 `WorldlineBranchCanvas.vue` 的画布氛围、边线层次和高亮效果。
- 强化了 `WorldlineBranchNode.vue` 的节点类型、当前激活态、选中态和不同路线气质。
- 当前分支已不再只是普通卡片，而是能形成“主问题 -> 世界线 -> 下一步”的分叉感。

### 3. 右侧面板层次

- 强化了 `WorldlineBranchDetailPanel.vue` 的信息层次与当前路线说明。
- 强化了 `WorldlineEvidenceRail.vue` 的证据解释表达。
- 强化了 `WorldlineNextStepPanel.vue` 的动作入口区分，让“进入对话 / 查看图谱 / 返回模块”更像下一步推进动作。

### 4. 工作台状态承接

- `worldlineContext` 继续保持轻量，不扩成重型状态机。
- 当前工作台已经能记住当前问题、当前分支、当前节点和承接动作提示。
- 从工作台进入 Agent / Graph 后，再返回工作台时，当前世界线状态不会立刻丢失。

## 本轮仍然刻意延后

以下能力仍明确延后，不属于当前第二波实现范围：

- 动态 AI 世界线生成器
- 正式世界线比较器
- 正式世界线切换器
- 世界线校正器
- 重型自动布局
- 拖拽式图编辑
- 全量知识导入
- 多智能体正式编排

## 验证基线

本轮稳定验证基线如下：

- `pnpm build` 通过
- `http://localhost:5173/themes/poe/worldline` 返回 `200`
- 浏览器内实际工作台页面渲染正常，用户可见中文正常显示
- 工作台页面首屏已经不是普通推荐卡页，也不是聊天界面
- 工作台能显示：
  - 顶部问题输入与引导区
  - 中部世界线分支舞台
  - 右侧详情 / 证据 / 下一步面板

## 阶段判断

- 当前阶段：`v1.2 实施`
- readiness：`ready 进入第三波“世界线生成感”增强或验收前整理`
- 当前最主要 gap：
  - 世界线工作台已经有舞台骨架，但还缺更强的“世界线生成感”
  - 分支树目前仍主要来自静态 PoE adapter，而不是更像实时生成的未来路径
  - 视觉已经明显提升，但还没有达到最终“高端、震撼”的成品感
